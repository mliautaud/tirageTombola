# tirageTombola
Cette page web permet de réaliser un tirage au sort pour une tombola.
2 CSV sont indispensable pour lancer le tirage : 
- les lots à gagner
- les tickets vendus pouvant être tirés au sort

Une fois les fichiers chargés, après chaque tirage au sort, le gagnant est affiché en surbrillance et persiste dans le tableau des gagnants.
Il n'y a pas de persistance des données.
On a la possibilité de relancer le tirage d'un lot. Le précédent gagnant est rayé car son lot est remis en jeu, et un nouveau tirage est réalisé.

Il va de soit que tous tickets ont la même probabilité d'être tirés au sort.

Bon jeu à tous ;-)
