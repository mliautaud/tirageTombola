# tirageTombola
Cette page web permet de réaliser un tirage au sort pour une tombola.<br>
2 CSV sont indispensable pour lancer le tirage :
<ul><li>les lots à gagner</li>
<li>les tickets vendus pouvant être tirés au sort</li>
</ul><br>
Une fois les fichiers chargés, après chaque tirage au sort, le gagnant est affiché en surbrillance et persiste dans le tableau des gagnants.<br>
Il n'y a pas de persistance des données.<br>
On a la possibilité de relancer le tirage d'un lot. Le précédent gagnant est rayé car son lot est remis en jeu, et un nouveau tirage est réalisé.<br>
<br>
Il va de soit que tous tickets ont la même probabilité d'être tirés au sort.<br>
